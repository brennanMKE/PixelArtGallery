# 0014 — Prevent importing duplicate images into the gallery

| | |
|---|---|
| **Status** | open |
| **Module** | ViewModels |
| **Platform** | All |
| **First seen** | 2026-06-25 |

## Description

The PRD lists duplicate prevention ("Option to skip importing images already in gallery") as a gallery storage feature. Import currently always creates a new `GalleryItem` with no de-duplication check.

## Expected behavior

On import, the app detects whether an identical image already exists (e.g. by content hash of the original bytes or dimensions+name) and offers to skip rather than creating a duplicate gallery item.

## Notes

- `PixelArtGalleryKit/Sources/PixelArtGalleryKit/ViewModels/GalleryCoordinator.swift` — `createGalleryItem`; add a hash/identity check before insert.
- Consider storing a content hash on `GalleryItem` to make the check cheap.
- PRD §1 "Duplicate Prevention". Depends on 0003 (real import path).
