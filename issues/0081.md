## Description

On macOS, FT **display management** (scan / add / edit / delete displays, in `DisplayRegistryView`) is only reachable through **Settings** (Pixel Art Gallery ‚ñ∏ Settings‚Ä¶, ‚åò,). There is no visible affordance in the main window, so users won't discover it. Add a **gear button to the macOS toolbar** that opens Settings, making display management obvious. (On iOS this is already covered ‚Äî the bottom bar has a Settings gear, #0071.)

## Notes

Relevant code:
- `PixelArtGalleryKit/Sources/PixelArtGalleryKit/UI/GalleryListView.swift` ‚Äî the `#if os(macOS)` toolbar currently holds only `+` (`.primaryAction`) and Sort (`.secondaryAction`); a comment there notes macOS reaches Settings via ‚åò, and "never had a Settings button here." - `PixelArtGallery/PixelArtGalleryApp.swift` ‚Äî the macOS `Settings { SettingsView() }` scene (‚åò,). `SettingsView` hosts `DisplayRegistryView` (display management, #0054).
- Mechanism: **`SettingsLink`** (SwiftUI, macOS 14+; the app targets macOS 15) opens the Settings scene natively and can be used directly as a toolbar item ‚Äî e.g. `SettingsLink { Label("Settings", systemImage: "gearshape") }`. Design points (planner to finalize ‚Äî macOS-only; do NOT change iOS, which already has the gear in its bottom bar):
- Add a **gear** toolbar item to the macOS `#if os(macOS)` toolbar in `GalleryListView` using `SettingsLink { Label("Settings", systemImage: "gearshape") }` (or a `Button` that triggers the Settings window if `SettingsLink` doesn't fit the placement). Keep ‚åò, working (the Settings scene stays).
- Placement/legibility: it joins the macOS toolbar alongside `+` and Sort ‚Äî coordinate with [#0080](0080.md) (which fixes that toolbar overlapping/illegible against the banner) so the gear is clearly visible too. - Optional: since the primary reason to open Settings is display management, consider whether the button should read as "Displays" ‚Äî but keeping it a standard Settings gear (that opens the Settings window where displays live) is the simplest, least surprising option; planner's call.
