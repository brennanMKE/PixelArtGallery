Version 1.0.0 Upcoming

Initial release — not yet published.

The first release of Pixel Art Gallery for macOS:

  • Gallery — import images into a persistent gallery presented as an adaptive thumbnail grid. Rename items, skip exact duplicates automatically, pin favorites so they always lead, and sort the rest by date or name.
  • Pixelation variants — create pixelated variants of any original at target dimensions from 1×1 to 512×512, with multiple variants per image. Duplicate variants or edit their dimensions after the fact.
  • Pixel preview — a zoomable, pannable pixel-grid preview with single-pixel selection showing exact coordinates and color values.
  • Export — save any variant as PNG, HEIC, PPM, or JSON pixel data.
  • Flaschen Taschen displays — discover LED displays on the local network via mDNS or add them manually by host and port; a default display is preconfigured. Send any variant straight to a display, with variants sized to match a display's native resolution.
  • Automatic updates — the app keeps itself current via Sparkle, checking this site's update feed.
