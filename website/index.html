Pixel Art Gallery

Turn any image into pixel art, organize it in a gallery, and light up LED displays with it.

Download for macOS

Version 1.0.0 · macOS 15 or later · Apple silicon & Intel · Free

From image to glowing pixels

Import a photo, pick a target size, and Pixel Art Gallery distills it down to clean, chunky pixels — ready to export or beam straight to an LED wall.

A real gallery

Every import lives in a persistent, adaptive grid. Pin your favorites so they always lead, and sort the rest by date or name.

Pixelation variants

Pixelate the same original at any target size from 1×1 up to 512×512 — keep multiple variants per image and compare them side by side.

Pixel-perfect preview

Inspect variants on a crisp pixel grid: zoom, pan, and click any single pixel to see its exact coordinates and color.

Export anywhere

Save variants as PNG or HEIC images, PPM bitmaps, or JSON pixel data — whatever your project needs.

Flaschen Taschen, built in

Discovers Flaschen Taschen LED displays on your network via mDNS (or add one by host and port) and sends variants with one click.

Always up to date

Built-in Sparkle updates check this site's release feed and install new versions securely — no app store required.

One original, many sizes

Each gallery item keeps its original image alongside every pixelated variant you create — 45×35 for the office LED wall, 32×24 for a test rig, 16×12 just for fun. Duplicate, resize, or delete variants without ever touching the original.

Preview, export, send

The variant view puts everything in one place: a zoomable, pixel-selectable preview, one-click export in four formats, and a send panel that pushes the artwork straight to the Flaschen Taschen display of your choice.

System requirements

Pixel Art Gallery is a native Mac app.

  • macOS 15 (Sequoia) or later
  • Apple silicon or Intel Mac
  • Optional: a Flaschen Taschen-compatible LED display on your local network
